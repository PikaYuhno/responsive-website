Tabellen in HTML


Tabellen werden in HTML mit dem <table> Tag, die Reihen der Tabellen werden mit dem <tr> Tag und die Tabellen Überschrift mit dem <th> Tag definiert. Die einzelnen Zellen mit Daten werden mit dem <td> Tag gekennzeichnet.


Tabellen in CSS formatieren

Mit dem der border Eigenschaft können Umrandungen dargestellt werden. padding bestimmt wie viel abstand zwischen den einzelnen Zellen gelassen wird. Die position des Textes kann mit text-align geändert werden. Außerdem kann mit colspan eine Zelle mehrere Spalten lang sein und mit rowspan eine Zelle mehrere Reihen hoch sein.
