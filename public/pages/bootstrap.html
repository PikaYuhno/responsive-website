Bootstrap


Bootstrap ist ein von Twitter eintwickeltes freies Frontend-CSS-Framework. Es enthält HTML und CSS design Templates für Typografie, Formulare, Tabellen, Grid-Systeme und Navigationselemente. Mit Bootstrap erstellte Elemente sind responsive. Dadurch werden die Seiten auf allen Bildschirmen schön angezeigt. Außerdem ist Bootstrap mit allen Browsern kompatiebel.
Bootstrap kann entweder von getbootstrap.com heruntergeladen werden oder ein CDN(Content Deliivery Network) wird verlinkt.


Grid-Systeme

In dem Grid System von Bootstrap gibt es bis zu 12 Spalten, welche individuell aufgeteilt werden. Die spalten sind dann auch responsive und werden sich neu anordnen abhängig von der Bildschirmgröße.
Es gibt fünf klassen für die Spalten: .col-, .col-sm-, .col-md-, .col-lg-, und .col-xl (von Klein nach Groß sortiert). Die Reihen müssen in einen .container oder einen .container-fluid stehen. Row erzeugt horizontale Reihen. Indem man nach .col- eine Zahl schreibt gibt man an wie viele der 12 verfügbaren Reihen man verwenden möchte.


Jumbotron

Jumbotron ist ein Interface-Element von Bootstrap und wird benutzt um Raum für den Inhalt der Webseite zu schafen. Es soll die Aufmerksamkeit der Benutzer gewinnen.
