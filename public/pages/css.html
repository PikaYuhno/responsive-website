CSS


Css steht für Cascading Stylesheets und ist eine Formatierungssprache, mit deren Hilfe Webseitenentwickler den Inhalt der Webseite von der Darstellung trennen können. Durch die CSS-Regel wird dem Browser mitgeteilt wie die HTML-Elemente formatiert werden sollen. Eine CSS-Regel besteht aus dem HTML-Element(Selektor) welches formatiert werden soll und seiner Formatierung(Deklarator). Der Deklarator besteht aus einer CSS-Eigenschaft und einem Wert, welche durch einen Doppelpunkt getrennt sind.


Arten von Selctoren

Universal-Selektor
Universal-Selektoren verknüpfen jedes vorhandene Element mit Stylesheetangaben.

Type-Selektoren
Typ-Selektoren verknüpfen einen Elementtyp, also ein HTML-Tag mit einer Formatierung.

Class-Selektor
Klassen sind Selektoren mit beliebigen Namen.

ID-Selerktor
ID-Selektoren verknüpfen Elemente über das ID-Attribut mit einer Formatierung.


CSS-Notation

Inline-Notation
Formatierung erfolgt in den HTML-Elementen oder ein Abschnitt wird mittels <span> formatieren. Jedoch wird die Inline-Notation sehr schnell unübersichtlich.

Zentrale Notation
Das Style-sheet wird zentral im head deklariert.

Ausgelagerte Notation
Das Stylesheet wird in einer externen CSS-Datei ausgelagert. Vorallem wichtig bei Webseiten die aus mehreren Webseiten bestehen, da die Formatierung dann für alle Seiten gleich ist. Das Stylesheet muss im head der Seite verlinkt werden.


Schriftarten

Die dem Browser, über CSS mitgeteilte Schriftart muss am System des Benutzers installiert sein, ansonnsten wird die Stadartschriftart(Times New Roman) verwendet. Die Schriftfarbe kann mit color geändert werden und die Hintergrundfarbe wird mit backround-color geändert. Hyperlinks haben vier verschiedene Zustände: noch nicht besucht(a: link), Links, über denen der Curser schwebt (a: hover), Gerade angeklickte Hyperlinks (a: active) und Besuchte Hyperlinks (a: visited). Alle vier zustände können vormatiert werden.
