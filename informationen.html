Algemeine Informationen zu HTML 5

HTML steht für Hypertext Markup Language. HTML verwendet das ASCII-Format und kann deswegen auf allen Computern verwendet werden. HTML ist eine Auszeichnungssprache(Beschreibungssprache für Dokumente), welche aus SGML(Standard Generalized Markup Language) enstand und im CERN entwickelt wurde. Das W3-Consortium ist für die Entwicklung verantwortlich.
