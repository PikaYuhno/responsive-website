HTML Aufbau


Tags

HTML Tags stellen die Befehle in HTML dar und bestehen aus einem öffnenden und einem schließenfen Tag. Selbstschließende Tags haben nur Atribute zwischen dem öffneneden und schließenden Tag. Manche Tags müssen bzw. dürfen nicht geschlossen werden.
Block-level Elemente(z. B. <div>, <table>, <p>) Erzeugen einen Zeilenumbruch, Inline Elemente(z. B. <span>, <table>, <img>) hingegen erzeugen keinen Zeilenumbruch. In einem Block-level Elements dürfen sich sowohl BLock- als auch Inline Elemente befinden. Jedoch dürfen in einem Inline Element nur weitere Inline Elemente sein.

Aufbau einer HTML Seite


1 <!DOCTYPE html>
2 <html>
3 <head>
4 </head>
5 <body>
6 </body>
7 </html>

Der Text wird zwischen die zwei body Tags geschrieben.
